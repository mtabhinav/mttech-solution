# MtTech Solution

MtTech Solution is a modern static landing page for a technology and digital services brand. The site presents MtTech's services, projects, workflow, and contact information through a polished single-page layout.

## Overview

This project is built with plain HTML, CSS, and JavaScript. It is lightweight, fast to load, and easy to deploy on any static hosting platform.

## Features

- Responsive single-page website
- Modern hero section and service showcase
- Project highlights and process section
- Smooth scrolling navigation
- Scroll reveal animations with `IntersectionObserver`
- Clean structure with no build step required

## Project Structure

```text
.
|-- index.html
|-- styles.css
|-- script.js
`-- README.md
```

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (`Outfit` and `Space Grotesk`)

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/mtabhinav/mttech-solution.git
```

2. Open the project folder:

```bash
cd mttech-solution
```

3. Run the website:

- Open `index.html` directly in your browser, or
- Use a local static server such as VS Code Live Server

## Deployment

This project can be deployed easily on:

- GitHub Pages
- Vercel
- Netlify
- Any static web hosting service

## Customization

- Update content in `index.html`
- Adjust visual styling in `styles.css`
- Modify reveal animation behavior in `script.js`
